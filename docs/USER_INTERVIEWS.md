# User Interview Synthesis

*Conducted: Q2 2024*
*Target Persona: CTOs, VPs of Eng, and Finance Leads at Series A-C startups.*

## Key Findings

### 1. The "Shadow IT" Problem
- **Observation**: Developers are expensing Claude Pro, ChatGPT Plus, and Cursor individually on personal corporate cards.
- **Quote**: "I looked at our Brex dashboard and realized we have 15 engineers expensing $60/mo each across 3 different tools. I don't even know if they use half of them." — *CTO, Series B Fintech*

### 2. The Enterprise Upsell Trap
- **Observation**: OpenAI and Anthropic heavily push Enterprise plans with high seat minimums (e.g., 150 seats). Startups buy these for "data privacy" but overpay massively.
- **Quote**: "We bought ChatGPT Enterprise for the SOC2 compliance, but honestly, only 20 people use it daily. We're paying for 150 seats." — *VP Eng, Series C SaaS*

### 3. Tool Overlap (The Copilot/Cursor redundancy)
- **Observation**: Engineering teams are migrating to Cursor, but GitHub Copilot subscriptions are left active because "it's part of the GitHub bill."
- **Quote**: "I didn't realize we were still paying $19/mo for Copilot for every dev while they were all using Cursor." — *Engineering Manager, Series A*

## Product Implications for Spendwise AI
1. **Show immediate dollar value**: The report must instantly calculate the "Shadow IT" and overlap waste.
2. **Focus on Security/Privacy**: The audit must reassure users that downgrading to API or Team plans doesn't necessarily compromise security.
3. **The "Hero" Moment**: Give the finance lead or CTO a report they can easily screenshot and drop in Slack to justify cutting redundant tools.
